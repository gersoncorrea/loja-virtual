# Loja Virtual

Construir uma aplicação web para loja virtual

## Requisitos

A aplicação deve atender os seguintes requisitos:
- Deve ser possível fazer o CRUD completo de produtos, contendo

> Tela de listagem

> Tela de cadastro/edição

> Exclusão de produtos
